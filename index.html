Webxdc.

A Web Without Platforms.

Webxdc is interactive content that is attached directly to a chat.

Privacy by definition: Webxdc just send messages to other members. Webxdc cannot load or transfer data to any servers.

Webxdc can be anything.

Collaborative Tools as Editors, Todo-Lists
Shared Calendars, Doodles
Polls
Games, of course

Try it out!

Download .xdc from below and attach it to any chat in Delta Chat 1.30 or newer.

For Developers.

A Webxdc is HTML5 content wrapped into a self-contained ".zip" named ".xdc".

Attached to a chat, you are already done.

Examples

You can find the examples from above at github.com/webxdc, improve them or use them as a template for your own, maybe far better ideas ❤️

If you prefer to start from scratch, you can clone the repo github.com/webxdc/webxdc-dev.
